### BUILDING AND RUNNING
##### CLIENT
```
make
```
```
./build/client/client <ip>
```
##### SERVER
```
make server
```
```
./build/server/server <local-ip>
```
#### DEPENDENCIES
You'll need raylib. \ 
You can install it on Arch Linux like this: 
```
sudo pacman -S raylib
```
Otherwise check your package manager or try building it yourself.
